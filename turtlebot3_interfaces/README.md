## Custom Interfaces

New custom messages and services defined for this project.

Can be imported like this: \
`from turtlebot3_interfaces.msg import ...` \
`from turtlebot3_interfaces.srv import ...`

### msg
- Mission
  - string mission_name
  - bool complete
- Progress
  - string sender
  - bool state
- Sign
  - string sign
  - int8 state
